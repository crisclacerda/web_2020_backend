# web_2020_backend

Rodar o projeto do backend simultanemaente com o do frontend

rodando o backend:
(lembrar de colocar o arquivo .env com as credenciais do banco de dados)
1- npm install
2- npm i axios --save
3- npm i cors --save
4- npm i nodemon --save
5- npm i bcrypt --save
6- npm install jsonwebtoken
7- npm run start

Acessar o site executando o projeto do frontend. NÃ£o fechar o terminal deste projeto (backend).
